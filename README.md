# Airman's Playbook

Public home for the **Airman's Playbook** and its associated team-facing
deliverables — a practical guide to help Airmen at all echelons adopt AI to improve
productivity and create mission capacity.

This repository publishes **only finished product**: the Playbook contents and the
artifacts we produce to present to the team. The private working/context layer that
informs this work is maintained separately and is intentionally not part of this repo.

## Layout
- `handbook/` — the Playbook itself (outline + sections + assets)
- `deliverables/` — team-facing artifacts (briefings, one-pagers, exports)
- `build/` — output pipeline (PDF / web), later

## Handling
**Non-sensitive content only.** Nothing controlled or sensitive belongs in this repository.
